# GitHub Profile Widgets Guide

This document explains the widgets used in the profile README and how to customize them.

## ðŸŽ¨ Widgets Included

### 1. Typing Animation Banner
- Service: [readme-typing-svg](https://readme-typing-svg.herokuapp.com)
- Displays animated cycling text at the top of your profile

### 2. GitHub Streak Stats
- Service: [github-readme-streak-stats](https://github-readme-streak-stats.herokuapp.com)
- Shows your contribution streak and statistics

### 3. GitHub Stats Card
- Service: [github-readme-stats](https://github-readme-stats.vercel.app)
- Displays comprehensive GitHub statistics

### 4. Top Languages
- Service: [github-readme-stats](https://github-readme-stats.vercel.app)
- Visual chart of your most-used programming languages

### 5. GitHub Trophies
- Service: [github-profile-trophy](https://github-profile-trophy.vercel.app)
- Showcases your GitHub achievements

### 6. Activity Graph
- Service: [github-readme-activity-graph](https://github-readme-activity-graph.vercel.app)
- Displays your contribution activity over time

### 7. Technology Badges
- Service: [shields.io](https://shields.io)
- Creates custom badges for technologies and tools

### 8. Social Badges
- GitHub followers badge
- Profile views counter via [ghpvc](https://github.com/antonkomarev/github-profile-views-counter)

## ðŸŽ¨ Theme

All widgets use the **Tokyo Night** theme for consistent styling.

To change the theme, replace `theme=tokyonight` with other options:
- `dracula`, `radical`, `merko`, `gruvbox`, `dark`, `synthwave`, etc.

## ðŸ”§ Customization

### Change Typing Animation Text
Edit the `lines=` parameter in the typing SVG URL:
```
lines=Your+Text+Here;Another+Line;Third+Line
```

### Add More Tech Badges
Visit [shields.io](https://shields.io) and generate badges for any technology.

### Adjust Language Count
Change `langs_count=8` in the Top Languages widget URL.
